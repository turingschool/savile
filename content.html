The s-content class can be used on any element where you want to style all the contents of that element. It will apply styles to the paragraphs using .s-p and links using .s-link, so you don't have to add those classes directly to every corresponding element.

{% assign s_content = site.content | first %}

HTML Example:

{% highlight html %}{{ s_content }}{% endhighlight %}
